# StreamTracker Web Service

**Project:** StreamTracker SaaS Platform  
**Created:** 2026-03-01  


## 🎯 Goal

Transform the local StreamTracker desktop app into a web-based SaaS service with:
- User accounts (Steam SSO)
- API key management
- Cloud-based achievement tracking
- Browser overlay (OBS integration)
- Premium features (Stripe billing)
